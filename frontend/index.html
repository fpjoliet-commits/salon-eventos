💡 ¿Cargando clientes de papel con fecha y pagos? ▾

Para un cliente que ya tiene fecha reservada y ya pagó algo, seguí este orden:

  1. Completá todos los datos del cliente (nombre, teléfono, tipo de evento, fecha, invitados, etc.)
  2. Poné Estado → Confirmado y Estado de la fecha → Tentativa Aunque ya esté reservada, primero hay que registrar la seña antes de poder marcarlo como Reservada.
  3. Guardá el cliente con el botón Guardar.
  4. Abrí la ficha del cliente que acabás de crear → tab Plan de pago → completá el presupuesto y el plan de cuotas.
  5. En "Registrar cobro", cargá el primer pago como tipo Seña. Los pagos siguientes como Cuota.
  6. Una vez registrada la seña, hacé clic en Editar → cambiá Estado de la fecha a Reservada → Guardar.

💡 ¿Cómo cargar el plan de pagos y registrar cobros? ▾

Parte de arriba — Plan de cuotas:

  1. Completá presupuesto total, moneda, cantidad de cuotas y fecha del primer pago → Guardar plan. El sistema genera las cuotas automáticamente.
  2. Si el cliente ya pagó cuotas, registralas con el formulario de abajo y tildá qué cuotas cubre ese pago.

Parte de abajo — Registrar cobro:

  1. Seña: primer pago del cliente. Al registrarlo podés cambiar el Estado de la fecha a Reservada.
  2. Cuota: tildar las cuotas del plan que cubre ese pago → se marcan como pagadas.
  3. Saldo final / Otro: solo registra el cobro, no toca el plan.
  4. La fecha del cobro es libre — podés poner cualquier fecha del pasado para cargar pagos históricos. Por defecto sale hoy.
